# 메이크로컬 TASK_LOG.md v0.1

## 문서 목적

이 문서는 메이크로컬 MVP 개발 과정에서 수행한 작업을 기록하기 위한 작업 일지다.

Codex 또는 개발자가 기능을 추가하거나 수정할 때마다 다음 내용을 기록한다.

- 작업 목표
- 수정한 파일
- 구현 내용
- 테스트 결과
- 남은 이슈
- 다음 작업

이 문서는 개발 과정에서 문제가 발생했을 때 어느 작업에서 변경이 있었는지 추적하기 위한 기준 문서로 사용한다.

---

## 작업 기록 작성 규칙

각 작업은 아래 형식으로 기록한다.

```txt
## YYYY-MM-DD 작업명

### 목표
- 이번 작업의 목표를 작성한다.

### 수정 파일
- 수정한 파일 경로를 모두 작성한다.

### 구현 내용
- 실제 구현한 내용을 작성한다.

### 테스트
- 테스트한 항목과 결과를 작성한다.

### 남은 이슈
- 아직 해결되지 않은 문제나 추후 확인할 내용을 작성한다.

### 다음 작업
- 이어서 진행할 작업을 작성한다.
```
